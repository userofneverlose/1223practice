# Use-Case диаграмма: Система «Обмен валют»

## Акторы
1. **Клиент API**: Любая программа или человек, отправляющий HTTP-запросы.
2. **Администратор**: Управляет справочной информацией (валюты и курсы).

## Таблица вариантов использования

| ID | Название | Описание |
| :--- | :--- | :--- |
| UC-01 | Получить список всех валют | GET /currencies |
| UC-02 | Получить одну валюту | GET /currency/{code} |
| UC-03 | Получить список всех курсов | GET /exchangeRates |
| UC-04 | Получить курс по паре | GET /exchangeRate/{pair} |
| UC-05 | Конвертировать сумму | GET /exchange?from=X&to=Y&amount=Z |
| UC-06 | Добавить новую валюту | POST /currencies |
| UC-07 | Добавить курс обмена | POST /exchangeRates |
| UC-08 | Обновить курс | PATCH /exchangeRate/{pair} |
